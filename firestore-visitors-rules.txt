Règles Firestore pour le compteur de visiteurs
==============================================
Dans la Console Firebase > Firestore > Règles, autorise lecture et écriture
sur le document stats/visitors. Exemple :

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Compteur de visiteurs : tout le monde peut lire et incrémenter
    match /stats/visitors {
      allow read, write: if true;
    }
    // Tes autres règles...
  }
}

Pour plus de sécurité en production, tu peux limiter l'écriture (ex. allow write: if request.resource.data.count == resource.data.count + 1).
